단순한 텍스트 파일(Markdown)을 아름답고 전문적인 PDF 문서로 출력하는 두 개의 유틸리티 웹 애플리케이션 파이프라인을 구축했습니다. **MD2PDF**(`08_md2pdf`)는 범용 문서 생성에, **MD_Resume**(`09_md_resume`)는 고품질 이력서 발간에 특화되어 있습니다.

<!-- more -->

## 📄 1. MD2PDF (08_md2pdf) 변환기
마크다운 문법으로 작성된 기술 문서나 기획안을 스타일이 입혀진 깔끔한 배포용 PDF 파일로 변환합니다.
*   **변환 프로세스**: `Flask` 백엔드가 사용자의 마크다운 입력을 받아 `markdown` 모듈로 먼저 HTML로 렌더링하고, 이후 CSS 스타일 시트(Github 테마 등)를 입혀 `WeasyPrint` 엔진에 넘겨 PDF를 생성합니다.
*   **한글 폰트 지원(Pre-embedded)**: 개발 초기 한글이 깨지는 현상이 있었으나, `fonts/` 디렉토리에 한국어 폰트(ex: Noto Sans KR 등)를 직접 임베딩하고 CSS의 `@font-face`를 지정하여 매끄럽게 문제를 해결했습니다. 
*   **코드 블록 하이라이팅**: `Pygments` 기반의 확장 기능을 활용하여 개발자들이 많이 사용하는 파이썬, 자바스크립트 등 다양한 소스 코드 블록의 시각적 형태를 그대로 유지한 채 출력해 냅니다.

## 💼 2. Markdown Resume Builder (09_md_resume)
위 엔진을 뼈대로 삼아 기능 특화시킨 '이력서/포트폴리오 생성기'입니다. 단순한 텍스트를 글로벌 IT 기업 스타일의 세련된 영문/국문 레주메 템플릿으로 탈바꿈시킵니다.
*   **다양한 디자인 테마 제공**: `static/css/` 안에 `theme_modern.css`, `theme_classic.css` 등 여러 개의 테마 CSS를 준비해두었습니다. 드롭다운에서 테마를 클릭하면, 마크다운 이력서가 전혀 다른 분위기(예: 깔끔한 모던 스타일 vs 중후한 클래식 스타일)로 즉각 변신하여 PDF로 나옵니다.
*   **이력서에 최적화된 마크다운 확장**: 경력(Work Experience)이나 프로젝트 타임라인 등을 수려하게 그리기 위해 특수한 마크다운 양식과 전용 CSS 클래스 맵핑을 구현하여 이력서 공간(여백, 페이지 분할 등)을 효율적으로 조율합니다.
*   **실시간 미리보기 (`app.py` 인터페이스)**: 작성한 마크다운을 브라우저의 양분된 화면 한쪽에 넣으면, 반대쪽에 HTML 수준에서 어떻게 렌더링 될지 실시간으로 보여주며(`example_resume.md` 기반), 완성 후 버튼 한 번이면 PDF가 즉시 다운로드 됩니다.

**💡 기술적 포인트**
두 애플리케이션 모두 복잡한 Word나 PPT 편집 없이 온전히 타이핑(코딩)의 효율만으로 정형화되고 일관된 산출물을 무한하게 만들어 낼 수 있는, 개발자 친화형 퍼블리싱 스위트(Publishing Suite)로 기능합니다.
