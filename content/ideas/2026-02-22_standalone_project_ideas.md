## 🎮 재미 / 개인 프로젝트

### 1. 타이핑 속도 테스트
- 랜덤 한글/영문 문장 → WPM 측정, 정확도 분석, 히스토리 그래프
- Flask + vanilla JS로 충분
- 난이도: ★★☆☆☆

### 2. 습관 트래커
- 매일 체크하는 습관 기록
- 연속 달성 streak, 히트맵(GitHub 잔디 스타일) 시각화
- 난이도: ★★☆☆☆

### 3. 밈 제너레이터
- 이미지 업로드 → 텍스트 오버레이
- Pillow로 서버사이드 합성, 갤러리 저장
- 난이도: ★★☆☆☆

### 4. 랜덤 메뉴 추천기
- "오늘 뭐 먹지?" → 위치/날씨/시간대 기반 메뉴 추천
- Gemini로 이유까지 설명
- 난이도: ★★☆☆☆

---

## 🛠 실용 / 유틸리티

### 5. 파일 정리 자동화
- 지정 폴더(다운로드 등) 감시 → 확장자/날짜별 자동 분류
- `watchdog` 라이브러리 활용
- 난이도: ★★☆☆☆

### 6. QR 코드 생성기
- URL/텍스트/WiFi 정보 입력 → QR 이미지 생성
- 커스텀 색상/로고 삽입 가능한 웹 UI
- 난이도: ★★☆☆☆

### 7. 마크다운 → PDF 변환기
- `.md` 파일 업로드 → 깔끔한 PDF 출력
- `weasyprint` 또는 `pdfkit` 활용
- 난이도: ★★★☆☆

### 8. 클립보드 히스토리
- 복사한 텍스트를 자동 저장, 검색 가능한 히스토리
- 시스템 트레이 상주 앱 (`pystray`)
- 난이도: ★★★☆☆

## 🤖 AI / API 활용

### 9. AI 면접관
- 직무/기술 입력 → Gemini가 면접 질문 생성
- 답변 입력 → 실시간 피드백
- 난이도: ★★★☆☆

### 10. 영어 문장 교정기
- 영작 입력 → Gemini가 문법 교정 + 자연스러운 표현 제안 + 난이도 표시
- 난이도: ★★☆☆☆

### 11. 유튜브 요약기
- YouTube URL 입력 → 자막 추출(`youtube-transcript-api`) → Gemini 요약
- 난이도: ★★★☆☆

### 12. AI 플래시카드
- 주제 입력 → Gemini가 Q&A 카드 자동 생성
- 간격 반복(Spaced Repetition) 학습 시스템
- 난이도: ★★★☆☆

## 🌐 웹 / 데이터

### 13. 단축 URL 서비스
- 긴 URL → 짧은 코드 생성
- 클릭 수/유입 경로 추적 대시보드
- 난이도: ★★★☆☆

### 14. 날씨 대시보드
- OpenWeather API → 현재 날씨 + 주간 예보 + 미세먼지
- 위젯 스타일 UI
- 난이도: ★★☆☆☆

### 15. 가격 변동 알리미
- 특정 상품 페이지 주기적 크롤링 → 가격 변동 시 알림
- 가격 히스토리 차트
- 난이도: ★★★★☆
