단기~중기 퀀트 투자 전술(스윙 트레이딩)을 누구나 쉽게 구사할 수 있게 해주는 개인용 대시보드 애플리케이션입니다. **Quant Dashboard**(`10_quant_dashboard`)는 방대한 주가 데이터와 수급 주체를 분석해 차트 밑에 찰떡같은 "기술적/펀더멘탈 분석 의견"을 달아줍니다.

<!-- more -->

## 📊 1. 퀀트 분석 모음집 (`analyzer.py`)
이 프로젝트의 핵심은 **단순한 선 긋기에서 벗어나 거래대금 폭발/고갈과 시장 강도를 다차원적 지표로 복합 판별**하는 강력한 파이썬 스코어링 모듈입니다.
*   **🔥 VCP 스윙 / MACD 스윙**: 에너지가 응축되어 변동성이 축소되다 거래대금이 폭발하며 21일 생명선을 돌파하는 정석 타점과, 보조 지표 추세를 통한 MACD 턴어라운드 반전 타점을 정확히 찾아냅니다. (보라색, 파란색 시그널)
*   **🚨 ATR Matrix 기반 극단 판별**: 고무줄처럼 주가의 변동성(ATR)을 측정, 50일 파란선을 기준으로 주가가 얼마나 이탈했는지 감시하여 `7 ATR 오버슈팅(과열 매수 금지)` 혹은 `-7 ATR 지하 암반수(투매 극단 저점)` 위치를 탐지해 냅니다.
*   **📐 피보나치 되돌림 분석기**: 장기 고점과 저점(150일)을 바탕으로 건강하게 숨을 고를 정석 1차 매수 타점(38.2%)부터 극단의 마지노선(61.8%)까지 스마트하게 선을 그어줍니다.

## 👥 2. 입체적 수급 파악 및 공매도 탐지 (`kis_api.py`)
한국투자증권(KIS) 실시간 정식 API를 전면에 배치하여 기관, 외인, 사모펀드 등의 세세한 돈의 흐름을 엑스레이처럼 투시합니다.
*   **쌍끌이 엔진**: 어제와 오늘 외국인/기관계/세부 기관(투신, 연기금 등)이 양쪽에서 강하게 매수해 주면 추가 점수를 대폭 가산하고, "🌍🏦 쌍끌이 매수! 가장 강력한 수급 신호" 라는 시그널을 출력합니다.
*   **진성 거래대금 파악**: yfinance의 추정치 대신 실제 실전 시장에서 호가를 오간 진짜 체결 단위별 일간 거래대금을 호출해와 평균 20일 거래대금과 비교합니다.
*   **공매도 잔고 어택 모니터링**: 숏 세력의 집중 어택을 받는 종목인지 공매도량 비중을 탐지하여 브레이크를 걸어줍니다(비중 5% 초과 시 강력 경고).

## 🖥 3. TradingView 차트 + 글래스모피즘 UI (`index.html`)
블랙과 네온 컬러의 아름다운 `Lightweight Charts`를 활용한 풀스택 모던 프론트엔드가 차트와 표면 지표를 동시에 띄워주는 미려한 우주선을 연상시킵니다. 우측의 '알고리즘 분석의견' 창이 주식을 매수해도 좋을지 100점 만점으로 점수를 매겨 실시간 피드백을 주며 하단에는 펼침/접기가 가능한 **지표 가이드북**이 동적으로 연동되어 있습니다.

**💡 배포 (`Render` 기반)**
별도의 인프라 구축 없이 Github Repository 푸시만으로 `Procfile`을 통해 포트 10000(또는 5000)번에 Gunicorn 프로덕션 서버로 바로 서빙될 수 있도록 클라우드 환경 설정(`render.yaml`)까지 마스터 피스로 구성되었습니다.
